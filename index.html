Simple Haskell Shell

This is a project to create a portable sh-like shell. One purpose is for writing portable test-suites that work the same on both unix and windows environments. A particular advantage is that shsh requires only a Haskell compiler to build on windows machines, and ultimately, we hope, will eliminate the need for cygwin/msys in many cases.

To get a current version of shsh, please run

  darcs get http://code.haskell.org/shsh
or download the tarball. We have both a franchise and a cabal build system (though cabal only builds the library). Patches are always welcome!

We also have a mailing list at http://groups.google.com/group/shsh-haskell for user or developer discussion.

language-sh

Language-sh is a haskell package that is co-maintained with shsh, containing anything pertaining to parsing or (eventually) printing shell scripts.

This can be downloaded as a separate library on Hackage, or otherwise it comes in the darcs repository or as a tarball
